# Assembly Line Stock · VSCode 盯盘插件

在 VSCode **状态栏**实时盯 A 股 / 场内 ETF / 指数 / 港股 / 美股 / 场外基金，支持**持仓盈亏、到价提醒、历史价格、搜索添加**，多窗口共享缓存、仅交易时段刷新。零运行时依赖。

状态栏效果（示意）：

```
$(graph) 总盈亏 +12345 (+8.23%)   茅台 1255.67 ▼1.21%   纳指广发 1.692 ▲0.89%   白酒基金 ≈0.5379 ▼1.67%
```

## 功能一览

| 功能 | 说明 |
|---|---|
| 多类型行情 | 股票 / ETF / 指数 / 港股 / 美股（实时价）、场外基金（净值，`≈`）；**自动显示中文名**（GBK 解码，港美股免别名）|
| **持仓盈亏** | 配成本价 + 持仓数，tooltip 显示每只浮盈，状态栏显示**总盈亏**汇总 |
| **到价 / 涨跌幅提醒** | 价格突破上/下限或涨跌幅达阈值时弹通知（防重复，条件解除后重置）|
| **搜索添加自选** | 输名字搜代码，免记代码、免手改 JSON |
| **历史价格** | 点状态栏标的 → Webview 看近 60 日收盘走势图 + OHLC 表 |
| 仅交易时段刷新 | A 股 / 港股 / 美股**各自**交易时段判断，A 股含**法定节假日** |
| 多数据源容错 | 腾讯为主，新浪为备，主源失败自动切换 |
| 多窗口共享缓存 | 开多个窗口出网请求 ≈ 单窗口 |
| **专属侧边栏** | 活动栏「盯盘」图标 → 自选面板：实时价、**目标价输入框**、拖拽排序、看历史、删除 |
| **LOF 双价** | 基金配 `lof:true`，状态栏显示场内实时价、tooltip 同时给场外净值 |

## 快速开始

```bash
npm install        # 安装开发依赖
# VSCode 打开本目录，按 F5 启动调试宿主窗口；或见文末打包安装到所有窗口
```

## 自选配置

`stockWatcher.symbols` 支持两种写法，可混用：

**① 简单写法**（字符串）：
```json
"sh600519:茅台"      // 代码:别名
"161725"             // 6 位纯数字 = 场外基金
"hk00700:腾讯"        // 港股
```

**② 进阶写法**（对象，带持仓 / 提醒）：
```json
{
  "code": "sz159941",
  "alias": "纳指广发",
  "cost": 1.50,        // 持仓成本价
  "shares": 1000,      // 持仓数量
  "alertAbove": 1.80,  // 价格 ≥ 1.80 提醒
  "alertBelow": 1.40,  // 价格 ≤ 1.40 提醒
  "alertPct": 5        // 当日涨跌幅 |≥5%| 提醒
}
```

代码前缀：上海 `sh`、深圳 `sz`、北京 `bj`、港股 `hk`、美股 `us`、场外基金为 6 位纯数字。

**LOF / 分级基金双价**：给基金加 `"lof": true`，插件会额外拉对应**场内实时价**——状态栏显示实时价（无 `≈`），tooltip 同时列出场内实时与场外估值净值。场内代码自动推断（5 字头沪市、其余深市），也可用 `"lofCode": "sz161725"` 显式指定。

**侧边栏面板**：点活动栏（最左图标栏）的「盯盘」📊 图标打开自选面板 —— 显示实时价 / 涨跌幅，每行有**目标价输入框**（输入数字回车即设提醒，按当前价自动判断"涨到 / 跌到"，清空回车取消）；可**拖拽 ⋮⋮ 调整顺序**、点名称看历史、× 删除；顶部有「添加自选 / 刷新」按钮。

> 不想手写？用命令 **`Assembly Line Stock: 添加自选（搜索）`**，输名字即可搜索添加。

## 命令（Cmd/Ctrl+Shift+P）

| 命令 | 作用 |
|---|---|
| `Assembly Line Stock: 立即刷新行情` | 强制刷新（无视时段与缓存）|
| `Assembly Line Stock: 添加自选（搜索）` | 输名字 / 代码搜索并加入自选 |
| `Assembly Line Stock: 删除自选` | 多选删除 |
| `Assembly Line Stock: 查看历史价格` | 看历史：**搜索任意标的**或从自选选；也可点状态栏 / 侧边栏标的 |
| `Assembly Line Stock: 开启/暂停盯盘` | 暂停 / 恢复刷新 |

## 持仓盈亏

给标的配 `cost`（成本价）+ `shares`（持仓数）后：

- 悬停该标的 → 显示「持仓 N @ 成本　浮动盈亏 +金额 (+%)」
- 状态栏出现 **总盈亏** 汇总项（所有持仓加总，红涨绿跌），可用 `showTotalProfit: false` 关闭

## 到价 / 涨跌幅提醒

**最简单**：在活动栏「盯盘」侧边栏，每只标的的**目标价输入框**输入数字回车即可（按当前价自动判断涨到 / 跌到）。

或在配置里配 `alertAbove` / `alertBelow` / `alertPct`：刷新时满足条件弹通知。**同一条件只弹一次**，价格回到条件外后自动重置，再次满足会再弹。

## 历史价格

**入口**：点状态栏 / 侧边栏标的；或命令「查看历史价格」——后者**支持搜索任意标的**（输入名称 / 代码即可，不必先加入自选）。打开 Webview 显示近期**走势图**（零依赖手绘 SVG，红涨绿跌）+ 明细表：

- A 股 / 场内 ETF / 指数：近 60 日**收盘价**走势 + 开高低收 / 成交量（新浪）
- 场外基金：近 30 日**单位净值**走势 + 每日净值 / 日增长（天天基金）
- 港股 / 美股：暂无可靠免费历史源，暂不支持

## 刷新行为

- **默认每 60 秒**自动刷新（`refreshInterval`，最低 5 秒）。
- **仅在所属市场交易时段自动刷新**：
  - A 股 / 基金：工作日 9:30–11:30、13:00–15:00，**已排除法定节假日**（含调休上班日，数据来自 holiday-cn，失败降级周末判断）
  - 港股：工作日 9:30–12:00、13:00–16:00
  - 美股：美东 9:30–16:00（精确夏令时：3 月第 2 周日 ~ 11 月第 1 周日）
  - 按**北京时间**判断（A/港股），不受电脑时区影响
- 非交易时段保留收盘数据，**手动刷新无视时段**。全天刷新可设 `onlyWhenMarketOpen: false`。
- **休市隐藏状态栏**：默认休市时段状态栏行情项自动隐藏（按各标的所属市场判断），活动栏侧边栏仍可查看；设 `hideStatusBarWhenClosed: false` 可让其常驻。
- **状态栏数量上限**：自选很多时状态栏只显示**前 5 个**（`maxStatusBarItems` 可调），超出部分显示为「+N」入口，点击打开侧边栏看全部——在侧边栏**拖拽排序**即可决定谁上状态栏。

## 配置项

| 配置 | 说明 | 默认 |
|---|---|---|
| `stockWatcher.symbols` | 自选列表（字符串或对象）| 见上 |
| `stockWatcher.refreshInterval` | 自动刷新间隔（秒，最低 5）| 60 |
| `stockWatcher.onlyWhenMarketOpen` | 仅交易时段自动刷新 | true |
| `stockWatcher.hideStatusBarWhenClosed` | 休市时隐藏状态栏行情项（侧边栏仍可看）| true |
| `stockWatcher.maxStatusBarItems` | 状态栏最多显示的自选数（超出显示「+N」，点击开侧边栏）| 5 |
| `stockWatcher.sharedCache` | 多窗口共享缓存去重 | true |
| `stockWatcher.showTotalProfit` | 状态栏显示总盈亏 | true |
| `stockWatcher.riseColor` / `fallColor` | 涨 / 跌颜色 | 红 / 绿 |

## 多窗口共享缓存

每个窗口是独立进程、各自刷新。插件用系统临时目录一份共享缓存去重：同一标的在一个刷新周期内只由一个窗口真正请求，其余窗口复用缓存 → 出网频率 ≈ 单窗口。原子写防并发损坏。关闭：`sharedCache: false`。

## 数据源

| 用途 | 数据源 |
|---|---|
| 实时行情（主）| 腾讯 `qt.gtimg.cn`（GBK，latin1 保字节）|
| 实时行情（备）| 新浪 `hq.sinajs.cn`（需 Referer，自算涨跌幅）|
| 场外基金净值 | 天天基金 `fundgz.1234567.com.cn`（UTF-8 JSONP）|
| 搜索 | 东方财富 `searchadapter.eastmoney.com` |
| 历史 K 线（A 股）| 新浪 `getKLineData` |
| 历史净值（基金）| 天天基金 `f10/lsjz`（需 Referer）|
| 法定节假日 | holiday-cn（jsdelivr CDN）|

> ⚠️ 场外基金盘中为**估算净值**，以当晚真实净值为准；行情仅供学习，不构成投资建议。

## 代码结构

```
src/
  types.ts       # 类型 + 市场判断（detectMarket）
  cache.ts       # 共享缓存 + 通用 JSON 文件缓存（原子写）
  market.ts      # 多市场交易时段 + 节假日（loadHolidays / isMarketOpen）
  providers.ts   # 行情(腾讯+新浪) + 基金 + 搜索 + 历史K线
  history.ts     # 历史价格 Webview（零依赖 SVG 走势）
  extension.ts   # 入口：命令、状态栏、刷新主流程、持仓盈亏、提醒
```
